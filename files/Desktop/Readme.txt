Welcome to WebOS 12

SYSTEM CONTROLS:
- Double-click desktop icons to launch applications.
- Use the Taskbar at the bottom to switch between open windows.
- Click the Clock area to view the Notification Center.

DEVELOPER TOOLS:
- Open the "Developer" app to view system architecture and kernel stats.
- Use the Terminal for low-level system commands.

PRO TIPS:
- You can install this OS as a PWA via your browser address bar.
- Right-click (or long-press) may reveal additional context menus.

Enjoy your refined web experience!
